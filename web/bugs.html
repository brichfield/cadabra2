Reporting bugs

Please report all bugs by email to kasper.peeters (at) aei.mpg.de, preferably with the shortest possible notebook which reproduces the problem. Please first check the list of known bugs.

A program of this scope and size is unlikely to be bug free, especially since the problems which it is meant to solve do not form a well-defined class. Most problems that have been reported so far are, however, easy to fix. Even if the program does something which you think is unexpected or undesirable but cannot really be classified as a bug, I would like to hear about it.
