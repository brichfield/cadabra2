Sample notebooks

This is a collection of sample notebooks for XCadabra. If you want to know how to handle a particular kind of problem for which there is no example available here, please post a message to the mailing list or email me directly.

These notebooks are served as application/cadabra; you can instruct your web browser to open these directly with xcadabra (usually located in /usr/bin/).

The PDF and Postscript files have been generated automatically by running LaTeX on the notebook file.

Examples from hep-th/0701238

The following notebooks are discussed in hep-th/0701238. Note: you may need cadabra version 0.118 or later for these to work.

  • Gamma matrix algebra: simple_clifford.cnb (or view as pdf or postscript)
  • Bianchi identities: bianchi_identities.cnb (or view as pdf or postscript)
  • Kaluza-Klein reductions: kaluza_klein.cnb (or view as pdf or postscript)

Other assorted examples

  • Instantons and self-duality: instantons.cnb (or view as pdf or postscript)
  • Commutators of the Poincaré algebra: commutators.cnb (or view as pdf or postscript)
  • Dot products: dot_products.cnb (or view as pdf or postscript)
  • Maxima interface (experimental): maxima.cnb (or view as pdf or postscript)
  • World-sheet supersymmetry: worldsheet_susy.cnb (or view as pdf or postscript)
