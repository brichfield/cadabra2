Download and install

Given the current state of Cadabra version 2, it is at present only distributed in source code form.

cadabra2-latest.tar.gz

You will need to install a number of dependencies before you can compile it. On Ubuntu/Mint/Debian simply do 

  sudo apt-get install cmake python-dev g++ libpcre3 libpcre3-dev libgmp3-dev 
  sudo apt-get install libgtkmm-3.0-dev libjsoncpp-dev libboost-all-dev libgmp-dev
  sudo apt-get install texlive python-matplotlib python-mpmath
  sudo apt-get install doxygen libjs-mathjax  
  sudo apt-get install libsqlite3-dev uuid-dev

You can get away with less (not all of boost is necessary, for instance), but the above should be sufficient to build everything, including the developers' documentation.

A git repository will be available shortly.
